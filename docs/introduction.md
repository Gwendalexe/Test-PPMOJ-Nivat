# 1. Introduction

Ce document a pour but d’expliquer le fonctionnement du site pour la prochaine personne qui va se pencher sur le sujet.

"Les Problèmes du Professeur Mojette”, a pour mission de dynamiser l'éducation scientifique à travers des jeux mathématiques captivants. Conçu par le Professeur Jean Pierre Guédon, notre plateforme aspire à renouveler l'attrait des lycéens pour les sciences, en offrant des énigmes originales et stimulantes. Porté par une équipe dédiée, l’objectif est de créer une expérience utilisateur immersive, accessible et éducative.

Trois jeux seront proposé sur le site, jeu mojette (Déjà disponible sur internet via <http://www.mojette.net/>  ce jeu est la première création de Jean-Pierre Guédon ), Carrés du Parterre et la Vadrouille en France  (étant le jeu principal sur le site il sera mis en avant sur la page d’accueil. Contenant 210 problèmes de type informatique, géométrique ou les deux réparti dans les régions et départements de France).
