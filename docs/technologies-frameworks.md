# 4. Technologie et Frameworks

- HTML : pour création et structuration de contenu des pages web.

- SCSS:  pour créatrion de style des pages web.

- Framework Bootstrap: pour création rapidement des interfaces web attrayantes et réactives.

- Framework Angular: pour construction d'application web dynamique et interactive côté client.

- TypeScript : pour l'ajout des fonctionalités aux pages web.

- Flask-Python: pour construction d'application web dynamique et interactive côté serveur.

- MySQL : pour stockage, organiseation et gestion des données.

- JQuery : Permet d'avoir la fonctionnalité drag and drop sur le jeu Carré par terre

- JWT : Protège avec un token les commandes sur le back

- Bcrypt: pour le hachage des mots de passes
